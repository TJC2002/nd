# ND网盘 - PC端页面规划

## 设计理念

遵循商业化系统级体验标准，采用扁平设计风格，支持 **5种主题**（dark暗黑、light明亮、白绿色自然、蓝白色、橙白色）切换，全页面融入流畅过渡动画（如hover微动效、页面切换淡入淡出、文件操作悬浮反馈），页面结构清晰且功能闭环。

## 主题系统

### 主题列表
1. **Dark（暗黑主题）** - 深灰背景，浅白文字，适合夜间使用
2. **Light（明亮主题）** - 浅白背景，深灰文字，适合日间使用
3. **白绿色自然主题** - 白色背景，深绿文字，图标以绿色为主，营造清新感
4. **蓝白色主题** - 白色背景，蓝色强调，专业商务感
5. **橙白色主题** - 白色背景，橙色强调，活力温暖感

### 主题切换
- 全页面颜色无缝过渡
- 插画元素随主题色自适应
- 切换时页面整体颜色渐变过渡

**Dark 主题**

## 交互设计原则

### 1. 动画效果
- **Hover 微动效**：按钮、卡片 hover 时有轻微缩放、颜色变化
- **页面切换**：淡入淡出过渡，避免生硬跳转
- **加载状态**：环形进度条、骨架屏加载
- **操作反馈**：成功/失败提示气泡，自动消失
- **拖拽效果**：半透明跟随、目标高亮闪烁

### 2. 响应式设计
- **布局适配**：支持 1366px、1920px、2560px 等主流分辨率
- **内容适配**：文件列表、卡片网格自动调整列数
- **字体适配**：支持系统字体缩放

### 3. 无障碍设计
- **键盘导航**：支持 Tab 键切换焦点
- **快捷键**：常用操作支持快捷键（Ctrl+C 复制、Delete 删除等）
- **屏幕阅读器**：语义化 HTML 标签，ARIA 属性完整

### 4. 性能优化
- **虚拟滚动**：大列表采用虚拟滚动，提升性能
- **懒加载**：图片、文件内容按需加载
- **缓存策略**：合理使用浏览器缓存，减少重复请求
- **动画优化**：使用 CSS transform 和 opacity，避免重排重绘
